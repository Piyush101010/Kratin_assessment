# Kratin_assessment

BOOKING CARE - Kratin Assessment Test

Booking Care is a web application for sunita,a 65+ years old woman to help her live healthy and happy life.

Features :
 1. Online Routine Checkup
 2. Diet Plan
 3. Exercise Suggestions
